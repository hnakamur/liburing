Detailed Description

This struct is used for parameters to io_uring_prep_openat2.

The value of this struct is processed in the build_open_flags function in the kernel.

uint64_t open_how::flags

flags for the openat operation.

The mask for valid values is VALID_OPEN_FLAGS whose value is (O_RDONLY | O_WRONLY | O_RDWR | O_CREAT | O_EXCL | O_NOCTTY | O_TRUNC | O_APPEND | O_NDELAY | O_NONBLOCK | O_NDELAY | __O_SYNC | O_DSYNC | FASYNC | O_DIRECT | O_LARGEFILE | O_DIRECTORY | O_NOFOLLOW | O_NOATIME | O_CLOEXEC | O_PATH | __O_TMPFILE)

◆ mode

uint64_t open_how::mode

mode for the openat operation.

When flags contains O_CREAT or __O_TMPFILE, the mask for valid values is S_IALLUGO whose value is S_ISUID|S_ISGID|S_ISVTX|S_IRWXUGO. When flags is 0, mode must be also 0.
